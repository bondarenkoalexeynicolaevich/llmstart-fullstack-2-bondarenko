# Задача 03: Аннотации в модели данных

**Итерация:** [iteration-1-scenarios](../../plan.md)

## Цель

Связать каждую сущность из `docs/data-model.md` с идентификаторами сценариев из [docs/spec/user-scenarios.md](../../../../../../spec/user-scenarios.md).

## Результат

- После заголовка каждой сущности (User, Flow, Participant, Module, Lesson, Assignment, Submission, DialogMessage) — блок-цитата со списком сценариев и ссылкой на спецификацию.
- При необходимости краткая строка в начале `data-model.md` со ссылкой на `docs/spec/user-scenarios.md`.

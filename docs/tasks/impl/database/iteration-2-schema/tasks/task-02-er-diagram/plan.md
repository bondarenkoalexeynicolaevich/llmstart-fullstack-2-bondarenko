# Задача 02: Физическая ER-диаграмма и DDL

**Итерация:** [iteration-2-schema](../../plan.md)

## Цель

В [docs/data-model.md](../../../../../../data-model.md) добавить раздел **Физическая схема PostgreSQL**: Mermaid-диаграмма (таблицы snake_case, FK, ключевые UNIQUE/индексы), полный DDL-скрипт, примечания по pgvector и отложенному IVFFlat.

## Шаги

1. Нарисовать физическую ER (включая `materials`, `knowledge_items`; VIEW — отдельным блоком или примечанием).
2. Вставить DDL согласованный с ревью (индексы на FK, составные индексы для истории/прогресса).
3. Уточнить: в коде enum через `native_enum`; в DDL документа — `TEXT + CHECK` как справочное представление.

## DoD

- Диаграмма и DDL согласованы друг с другом и с логической моделью.

# task-03: чеклист перед коммитом

## Что сделать

- **Минимум:** в README (в разделе качества или отдельным подпунктом) зафиксировать чеклист: **`make lint`**, **`make test`** перед коммитом.
- **Опционально:** `.pre-commit-config.yaml` с ruff — только если команда готова поддерживать hooks; в этой реализации приоритет — чеклист в README без обязательного pre-commit.

## Критерий готовности

- В README явно перечислены команды, которые стоит выполнить локально перед коммитом.
